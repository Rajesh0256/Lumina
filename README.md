# Lumina — Flask Social Media App

## Setup

```bash
cd lumina
pip install -r requirements.txt
python app.py
```

Then open http://127.0.0.1:5000

## Features
- Register / Login
- Upload photos with caption & location
- Like / Unlike posts
- Comment on posts
- Follow / Unfollow users
- Personalized feed (posts from followed users + your own)
- Explore page
- Profile grid view
- Delete your own posts
